2.2 Récupération de données provenant du web : le web scraping

Le web scraping est une technique permettant d’extraire, le plus souvent de façon automatique, le contenu d’une page web. Cela peut être très utile pour récupérer des données publiques nécessaires à nos analyses.

Sous R, le package permettant de parcourir et de récupérer facilement le contenu d’une page web s’appelle rvest.

library("rvest")

Ici, mon but est de récupérer la classification des différentes espèces d’oiseaux de la Réserve Naturelle Nationale de la Haute Chaîne du Jura (tableau 2.5). Cette classification peut être facilement récupéré sur les pages Wikipédia des oiseaux en question. Je veux donc créer une fonction qui va rechercher la page Wikipédia de l’oiseau voulu, puis récupérer sous la forme d’un tableau sa classification.

get_bird_classification <- function(bird, taxon = c("Ordre", "Famille")) {
  wiki_link <- "https://fr.wikipedia.org/wiki/Wikip%C3%A9dia:Accueil_principal"
  session <- html_session(wiki_link)
  
  form <- html_form(session)
  form_modified <- set_values(form[[1]], search = bird)
  
  # On entre sur la page wikipédia de l'oiseau recherché.
  wiki_bird <- submit_form(session, form_modified, submit = "go")
  
  tbl_classification <- wiki_bird %>%
    html_node(xpath = "//table[@class='taxobox_classification']") %>%
    html_table() %>%
    magrittr::set_colnames(c("niveau_classification", "nom_classification")) %>%
    filter(niveau_classification %in% taxon) %>%
    tidyr::pivot_wider(
      names_from = niveau_classification,
      values_from = nom_classification
    )
  
  tbl_classification$nom <- bird
  
  return(tbl_classification)
}
noms_latin <- stringr::str_extract_all(avifaune_jura$nom_latin, r"(^\w+\s+\w+)")

classification_oiseaux <- purrr::map(noms_latin, get_bird_classification) %>%
  bind_rows() # Fusion des tableaux
avifaune_jura <- avifaune_jura %>%
  mutate(nom = unlist(noms_latin)) %>%
  left_join(classification_oiseaux, by = "nom") %>%
  select(-nom) %>%
  relocate(Ordre:last_col())
